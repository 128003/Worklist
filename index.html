# 復健科臨床工具箱 評估量表、決策樹與臨床計算器集合，全部資料皆儲存在使用者瀏覽器本機（localStorage），不會上傳到任何伺服器。 ## 📁 資料夾結構 ``` rehab-toolkit/ ├── index.html ← 工具箱首頁 ├── assets/ │ ├── style.css ← 共用樣式 │ └── common.js ← 共用儲存/複製/歷史紀錄函式 └── tools/ ├── mmt-score.html ├── rom-goniometer.html ├── ottawa-rules.html ├── outcome-scales.html ├── special-tests.html ├── concussion-track.html ├── cait-ankle-instability.html ├── gird-shoulder.html ├── spadi-shoulder.html └── visa-a-achilles.html ``` ## 🚀 部署到 GitHub Pages（步驟） 1. 到 GitHub 建立一個新的 repository（例如取名 `rehab-toolkit`），設為 Public。 2. 把這個資料夾內的所有檔案（保持原本的資料夾結構）上傳到該 repository 的根目錄： - 網頁版：直接在 repo 頁面點「Add file → Upload files」，把整個資料夾拖進去 - 或用 git 指令： ``` git init git add . git commit -m "initial commit" git branch -M main git remote add origin https://github.com/你的帳號/rehab-toolkit.git git push -u origin main ``` 3. 到 repository 的 **Settings → Pages**。 4. 在「Build and deployment」的 Source 選擇 **Deploy from a branch**，Branch 選 `main`，資料夾選 `/ (root)`，按 Save。 5. 等待 1–2 分鐘，網址會顯示在同一頁面，格式通常是： `https://你的帳號.github.io/rehab-toolkit/` 6. 之後每次更新檔案、push 上去，網站會自動重新部署。 ## 📱 之後要加 QR Code 給病人自填 部署完成後，把任一工具的完整網址（例如 `https://你的帳號.github.io/rehab-toolkit/tools/spadi-shoulder.html`）貼到首頁的 QR Code 產生器欄位，就能產生對應 QR code 給病人掃描。 ## ⚠️ 注意事項 - 所有量表分數計算僅供臨床教育輔助參考，不可取代專業判斷。 - QuickDASH / KOOS / HOOS / WOMAC / CAIT / SPADI / VISA-A 等量表的「官方題目文字」屬原作者版權，本工具僅提供計分公式換算，正式臨床使用請搭配院內授權之正式量表本。 - 請勿在任何頁面輸入可識別病人身分的真實資料；所有紀錄僅存於使用者當下的瀏覽器，清除瀏覽器資料或換裝置會遺失。
